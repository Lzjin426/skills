# Profile: Proposal

## When to use

Use this profile for recommendations, option comparison, decision support, and technical or product proposals.

## Reader expectation

The reader expects a recommendation, the reasoning behind it, and clear trade-offs.

## Default structure

- problem or decision context
- options
- comparison criteria
- recommendation
- risks, consequences, or next steps

## Default carrier

Paragraph-led with selective comparison tables.

## Default passes

- research: `optional`
- cognitive-path: `off`
- illustration: `optional`
- humanization: `off` by default

## Style defaults

- be explicit about judgment and trade-offs
- separate evidence from opinion
- keep the recommendation easy to find
