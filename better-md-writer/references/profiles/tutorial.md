# Profile: Tutorial

## When to use

Use this profile for guided learning, concept teaching, or step-by-step walkthroughs where understanding matters, not just completion.

## Reader expectation

The reader expects help building understanding and then using it successfully.

## Default structure

- what the reader will learn
- required context or prerequisites
- guided explanation and steps
- verification or expected outcome
- next extensions or common pitfalls when useful

## Default carrier

Paragraph-led with step lists and code blocks.

## Default passes

- research: `off`
- cognitive-path: `optional`
- illustration: `optional`
- humanization: `optional`

## Style defaults

- teach before or alongside commands, not after them
- establish a mental model only when the tutorial is long enough to benefit from it
- keep the reader moving forward with clear checkpoints
