# 本地周报模板（Markdown）

```markdown
---
tags:
aliases:
---
# {{M.DD～M.DD-YY}}

## 主要完成工作

### {{按当周主线动态命名的小节}}
{{先用 1 段话概述本周该主题的整体推进，再补 2~4 条关键点。}}
- {{关键推进点 1}}
- {{关键推进点 2}}

### {{如有第二主线，再增加小节；没有就不要硬凑}}
{{仍然保持“概述 + 适量要点”的写法。}}
- {{关键推进点 1}}

## 关键阶段性成果
- {{成果 1：产出 + 价值}}
- {{成果 2：产出 + 价值}}

## 存在问题与风险
- {{问题 1：现状 + 影响 + 下一步}}
- {{问题 2：现状 + 影响 + 下一步}}

## 下周工作计划
- {{计划 1}}
- {{计划 2}}
- {{计划 3}}
```

## 使用提醒

- 文件名使用 ISO 区间：`YYYY-MM-DD～YYYY-MM-DD.md`
- 正文标题使用旧标题格式：`# M.DD～M.DD-YY`
- `主要完成工作`中的三级标题数量按内容决定，不追求固定数量
- 缺失日报只做事实说明，不作推断
