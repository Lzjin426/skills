# 齿轮接触应力校核工具 V1.0 用户手册

## 3.2 接触应力计算模块

### 3.2.1 功能概述

本模块旨在为用户提供一套完整、专业的齿轮接触应力计算解决方案。通过内置的多种
计算算法和校核标准，用户可以针对不同的工程应用场景选择合适的计算方法，从而
实现对齿轮接触应力的高效、精确评估。本模块不仅支持基于 ISO 6336 标准的经典
Hertz 接触应力计算，而且支持多种压力角和材料参数的灵活配置，具有灵活性高、
易用性强、计算精度高的特点。

本模块广泛适用于各类工业齿轮设计场景，能够极大地提升工程师的设计效率，显著
降低设计周期，全方位赋能齿轮传动系统的研发工作。

### 3.2.2 操作步骤

用户可以通过以下步骤使用本模块进行接触应力计算：

1. 首先，用户需要在主界面点击"接触应力计算"按钮，进入本模块的操作界面
2. 其次，用户需要在参数配置面板中填写小齿轮的各项参数，包括模数、齿数、齿宽、
   弹性模量以及泊松比
3. 然后，用户需要以同样的方式填写大齿轮的各项参数
4. 接下来，用户需要在传动参数面板中填写传递转矩以及压力角
5. 最后，用户只需点击"开始计算"按钮，系统将自动执行计算流程，并在结果面板中
   展示最终的接触应力值

### 3.2.3 界面说明

本模块的主要界面由以下几个部分组成：

- **参数输入区**：位于界面左侧，用于输入小齿轮和大齿轮的各项参数
- **传动参数区**：位于界面中部，用于输入转矩和压力角
- **结果展示区**：位于界面右侧，用于展示计算结果
- **操作按钮区**：位于界面底部，包含"开始计算"、"清空"、"导出"等按钮

### 3.2.4 注意事项

在使用本模块时，用户需要注意以下几点：

- 所有的参数值必须为正数，否则系统将提示错误信息
- 压力角通常取 20° 或 25°，建议用户根据实际齿轮标准进行选择
- 计算结果仅供参考，在实际工程应用中应结合具体工况进行综合评估
- 本模块的计算基于理想条件假设，未考虑齿轮制造误差、安装误差等实际因素

### 3.2.5 计算结果解读

计算完成后，系统将在结果展示区展示以下信息：

- **接触应力值**：以 MPa 为单位显示的 Hertz 接触应力计算结果
- **安全系数**：基于用户输入的许用应力自动计算得出的安全系数
- **评估结论**：根据安全系数给出"安全"或"不安全"的评估结论

综上所述，本模块为用户提供了一个功能完整、操作便捷、结果可靠的接触应力计算
工具，能够有效地助力工程师完成齿轮设计工作。

## 3.3 参数优化模块

### 3.3.1 功能概述

本模块致力于为用户提供基于多种优化算法的参数寻优能力。通过集成遗传算法（GA）、
非支配排序遗传算法（NSGA-II）以及粒子群优化算法（PSO）等先进的优化算法，用户
可以在给定的参数搜索空间内高效地寻找最优参数组合。本模块具有算法多样、
配置灵活、结果可视化等显著优势。

### 3.3.2 操作步骤

1. 首先，在主界面点击"参数优化"按钮进入本模块
2. 其次，在参数配置界面设置待优化参数的取值范围和约束条件
3. 然后，选择合适的优化算法并配置相应的算法参数
4. 最后，点击"开始优化"按钮，等待系统完成优化计算

### 3.3.3 注意事项

- 优化过程可能耗时较长，请耐心等待，切勿中途关闭程序
- 建议合理设置种群大小和迭代次数以平衡计算效率与优化质量
- 多目标优化场景下推荐使用 NSGA-II 算法
