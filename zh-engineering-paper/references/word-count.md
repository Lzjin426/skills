# 字数 / 篇幅规范速查

中文工科核心 / 顶刊的字数要求散落在各期刊投稿指南里，本表汇总常见区间。**真正投稿前请按目标期刊指南二次核对**——例如《机械工程学报》和《自动化学报》对摘要字数的要求并不完全一致。

## 一、整体字数

| 项 | 中文核心常见区间 | 顶刊常见区间 | 说明 |
|---|---|---|---|
| 全文（不含参考文献）| 6000–10000 字 | 8000–14000 字 | 顶刊鼓励"讲透"，篇幅可宽 |
| 参考文献条数 | 25–40 | 35–60 | 综述更多，可达 80+ |
| 全文图表数（含算法/伪代码）| 6–12 | 8–18 | 顶刊允许更多 |

## 二、各章节字数

| 章节 | 字数 | 占全文比例 | 关键要求 |
|---|---|---|---|
| 标题（中文）| **16–28 字** | — | 三层信息：方法 + 对象 + 场景 |
| 标题（英文）| 12–20 词 | — | 与中文意义对应 |
| 摘要（中文）| **250–400 字** | 3–5% | 250 是最低线，多数期刊 300 字左右最佳 |
| 摘要（英文 Abstract）| 180–280 词 | — | 与中文严格对应 |
| 关键词 | **5–8 个** | — | 3 个领域大词 + 3 个具体方法/对象 + 1–2 个差异化 |
| 引言 | **1500–3000 字** | 12–18% | 4 段式（领域定义 / 现状 1 / 现状 2 / 空白+创新点）|
| 问题描述 / 基础理论 | 500–1500 字 | 5–10% | 形式化 + 必要预备知识 |
| 方法（每个创新点 1 章）| **1000–1800 字 / 章** | 合计 35–45% | 动机段 + 设计 + 公式 + 性质 |
| 实验 | **2000–3500 字** | 25–35% | 设置 / 主实验 / 消融 / 可视化 |
| 结论 | **300–600 字** | 4–6% | 工作概述 + 量化成果 + 应用价值 + 不足展望 |

## 三、关键段落字数

引言 4 段的内部分配：

| 段位 | 字数 | 说明 |
|---|---|---|
| 第 1 段：领域定义 + 脉络 | **200–300 字** | 1 个奠基方法 + 2–3 个子方向 |
| 中段：现状梳理 | **800–1500 字** | 按子话题切分，每段 200–400 字 |
| 末段：空白 + 创新点 | **300–500 字** | 创新点 (1)(2)(3) 占大头 |

创新点表达在三处的字数（同一条创新点）：

| 出现位置 | 字数 | 视角 |
|---|---|---|
| 摘要内 | **30–50 字** | "提出 / 设计了 X" |
| 引言末段 (1)(2)(3) | **80–120 字 / 条** | "做法 + 机理收益" |
| 结论内 | **40–80 字 / 条** | "做法 + 验证结果" |

## 四、句长

| 项 | 推荐区间 | 警戒线 |
|---|---|---|
| 单句字数 | 20–60 字 | > 80 字必须拆 |
| 段落首句字数 | 15–30 字 | 段首句子要清爽 |
| 一段字数 | 100–300 字 | > 400 字考虑拆段 |
| 一段句数 | 3–6 句 | > 8 句必拆 |

## 五、图表

| 项 | 推荐 | 注意 |
|---|---|---|
| 单图说明（caption）字数 | 5–25 字 | 子图说明用 (a) (b) (c) 标注 |
| 表格行数 | 4–15 行 | > 20 行考虑拆表或转图 |
| 表格列数 | 3–8 列 | 太宽要横排或转置 |

## 六、不同子领域的微调

中文工科顶刊的实际字数会因子领域而异：

- **力学 / 材料 / 拓扑优化**（如样本中蔡金虎论文）：偏短，6000–9000 字常见，公式多、文字密
- **AI / 控制 / 故障诊断**（如样本中张九思、朱显丞论文）：偏长，9000–13000 字，因为要详述网络结构、消融实验
- **机械结构 / 风电 / 流体**（如样本中李浩、巩加玉论文）：图表多，8000–12000 字
- **综述类**：篇幅最长，12000–20000 字，可达 60+ 篇参考文献

## 八、字数不达标的常见后果

| 情况 | 后果 |
|---|---|
| 摘要太短（< 200 字）| 编辑可能直接退回；信息密度不够 |
| 摘要太长（> 500 字）| 部分期刊系统会截断或编辑要求精简 |
| 引言太短（< 1000 字）| 现状梳理不充分，审稿人怀疑文献调研深度 |
| 引言太长（> 4000 字）| 喧宾夺主；审稿人会要求精简到方法/实验 |
| 结论太长（> 1000 字）| 像第二个摘要；要求合并到讨论 |
| 全文太短（< 5000 字）| 顶刊基本不接收，建议改投快报或 letters |
| 全文太长（> 18000 字）| 排版页数超限；审稿人疲劳；考虑拆成两篇 |

## 自查清单

- [ ] 标题字数在 16–28 字
- [ ] 摘要字数符合目标期刊要求
- [ ] 关键词 5–8 个，类型分布合理
- [ ] 引言 4 段式，各段字数符合分配
- [ ] 创新点 (1)(2)(3) 在摘要 / 引言末段 / 结论三处都出现，字数对应
- [ ] 单句不超过 80 字
- [ ] 单段不超过 400 字
- [ ] 全文字数符合目标期刊
