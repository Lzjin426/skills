# 标杆论文仿例

本 reference 选自 2 篇真实顶刊论文（《机械工程学报》），把**摘要 / 引言 / 结论**完整呈现，并在每段后用 `→` 标注**段落功能**和**值得模仿的写作动作**。

写作时，对照仿例的"节奏"，比对照零散规则更直观——审稿人感受到的是整体节奏。

## 仿例选择说明

- **仿例 A：蔡金虎《动态载荷作用下的双模量材料结构拓扑优化》** —— 力学 / 拓扑优化方向，结构最为经典，引言四段式与创新点表达堪称模板
- **仿例 B：张九思《半监督学习视角下序列化变分自编码器驱动的高端装备智能剩余寿命预测》** —— 深度学习 / 智能制造方向，覆盖与仿例 A 不同的子领域

两篇都来自同一份顶刊（《机械工程学报》），DOI 和期刊信息均真实，下载于本 skill 创建时的样本库。

> **重要免责声明**：仿例供**模仿写作风格、节奏、句式**，**绝不可抄袭**。摘要、引言、创新点的具体内容是原作者的研究成果，未经授权不得复制使用。

## 仿例 A：蔡金虎《动态载荷作用下的双模量材料结构拓扑优化》

### A.1 标题分析

> **动态载荷作用下的双模量材料结构拓扑优化**

→ **结构**：「工况（动态载荷作用下）」+ 「对象（双模量材料结构）」+ 「任务（拓扑优化）」
→ **字数**：18 字
→ **关键词位置**：核心方法（拓扑优化）放在末位，强调"做了什么任务"；工况放在最前以突出新意（"以前研究没考虑动态载荷"）
→ **可模仿点**：当你的工作核心是"在新工况下做某个老任务"时，这种结构最合适

### A.2 摘要（共约 280 字）

> 实际工程中的很多材料在拉伸和压缩状态下具有不同的力学性能（即双模量特性）。

→ **【背景】1 句 33 字**：直接定义研究对象，**没有"在当今""随着...的发展"**
→ **可模仿点**：第一句不出现"本文""我们"，让领域陈述自己说话

> 然而，针对这类材料结构的拓扑优化方法研究很少，尤其是极少考虑动态载荷下的双模量材料结构设计。

→ **【问题】1 句 39 字**：用"然而"转折；指出**两层不足**（双模量整体研究少 + 动态载荷下更少）
→ **可模仿点**：问题段写出"嵌套式不足"，把审稿人引到"必须做这件事"的心理位置

> 因此，提出了一种动态载荷下的双模量材料结构拓扑优化方法。首先，采用混合应力单元离散设计域，以获得更准确地应力计算结果；其次，建立基于单元主应力符号和拉/压容限的单元拉/压状态判断准则，构建了四相材料插值模型实现双模量材料结构优化设计；接着，发展多工况载荷下基于单元拉/压应变能的单元拉/压状态判断准则，实现了多工况载荷下的材料分布优化；最后，通过典型数值算例验证了方法的有效性，并分析了拉/压弹性模量比值、激励频率等参数对优化构型的影响规律，

→ **【方法】递进 4 步 175 字**："因此 → 首先 → 其次 → 接着 → 最后"
→ **可模仿点**：每一步都用"动作 + 目的/收益"结构（"采用混合应力单元 → 以获得更准确..."），不是只列动作
→ **方法步骤分量分配**：方法主体（首/其/接）3 步占 75%，验证（最后）占 25%

> 为双模量材料结构在动态载荷下的轻量化设计提供理论支撑。

→ **【意义】1 句 25 字**："为...提供理论支撑"是顶刊摘要末句的高频固定句式
→ **可模仿点**：意义段不空喊"具有重要意义""有广阔前景"，而是定向说"为 X 提供 Y"

**摘要全文统计**：约 280 字（落在 250–400 区间），方法占 60%，背景+问题占 25%，意义占 5%

### A.3 引言（共 4 段，约 1100 字）

#### 第 1 段：领域定义 + 学术脉络（约 200 字）

> 拓扑优化作为一种结构轻量化设计方法，旨在通过优化材料分布，获得给定约束条件下的最优拓扑构型 [1]。自 BENDSØE 和 KIKUCHI 提出均匀化方法 [2] 以来，该领域研究范畴已从传统的柔度优化拓展至多物理场耦合问题。学者们在热力耦合拓扑优化 [3]、装配结构系统优化 [4] 等方向取得显著进展；同时，面向工程应用的大规模并行计算框架 [5] 和考虑增材制造工艺的多尺度结构设计 [6] 的发展，进一步推动了该领域的工程实用化进程。

→ **段落内部结构**（4 句）：
1. 句 1：领域定义（"X 作为 Y，旨在 Z [1]"）
2. 句 2：奠基性工作（"自 ... 提出 ... [2] 以来，该领域已从 A 拓展至 B"）
3. 句 3：列举子方向（"在 ... 等方向取得显著进展"）
4. 句 4：工程化趋势（"... 推动了该领域的工程实用化进程"）

→ **关键句式**："...作为...，旨在通过...，获得..." —— 领域定义的标准模板
→ **可模仿点**：第 1 段**不引出问题**，纯粹搭建领域大背景；问题留到第 3 段。这样让现状梳理（第 2 段）有空间自然展开。

#### 第 2 段：现状梳理（双模量材料的相关工作，约 280 字）

> 在工程实践中，大量的结构材料（如钢材、混凝土和纤维增强复合材料 [7] 等）表现出显著的双模量效应——即在拉伸和压缩载荷作用下呈现不同的弹性模量，这类具有拉/压非对称特性的材料通常被定义为双模量材料。为准确表征其非线性本构关系，CAI 等提出了材料置换法：使用两种拉/压性能不同的各向同性材料替换原来的双模量材料 [8-9]。在数值计算层面，DU 等 [10] 基于 SIMP 插值法，创新性地推导了单/双相双模量材料的严格灵敏度分析结果，并开发出具有数值稳定性的高效优化求解器。LI 等 [11-12] 提出了材料利用率和主应力之和准则分别用来判断单元中填充的材料对结构的重要性以及单元所处的拉/压状态，并研究了由多相拉伸材料和压缩材料组成的复合材料结构拓扑优化问题。JIANG 等 [13] 针对考虑位移约束的双模量材料属性不光滑带来的数值困难问题，构建了新的光滑弹性模量矩阵，并基于此建立了单/多载荷工况下的拓扑优化模型。针对连续纤维增强复合材料的拉/压非对称特性，LIU 等 [7] 创新性提出多尺度协同优化框架，实现了宏观拓扑构型、纤维取向分布及纤维体积分数的并发拓扑优化设计。GUO 等 [14] 则针对纤维增强复合材料的可变刚度设计，提出了一种类似于桁架结构的拓扑优化方法。

→ **段落内部结构**：
1. 概念铺垫（"在工程实践中，大量的结构材料...被定义为双模量材料"）
2. 引出多个相关工作：CAI 等 → DU 等 → LI 等 → JIANG 等 → LIU 等 → GUO 等

→ **关键句式**：
   - "XX 等提出了 ... [N]"（最简单）
   - "XX 等 [N] 基于 X 方法，创新性地 Y，并 Z"（介绍贡献+方法）
   - "XX 等 [N] 针对 ... 问题，构建了 ..."（介绍问题+方法）
→ **可模仿点**：
- 同一段内对**多个相关工作**做平行罗列，但**句式略有变化**避免单调
- 每个工作 1–2 句话；不评价、不展开

#### 第 3 段：现状梳理（动态载荷下的相关工作 + 引出空白，约 380 字）

> 然而，上述有关双模量材料结构的拓扑优化研究只考虑了静态载荷条件下的结构响应问题，鲜有涉及动态载荷作用下的优化设计。值得注意的是，在航空航天等实际工程领域中，飞行器结构常承受发动机振动、气动湍流等动态载荷的持续作用，对系统的安全性和可靠性具有决定性影响。针对时域载荷作用下的动态拓扑优化研究：JANG 等 [15] 基于等效静态载荷方法构建了时域动态响应拓扑优化框架；JIANG 等 [16] 面向多晶胞双尺度结构的时域动刚度最优设计问题，考虑不同晶胞间的可连接性，并行设计了微结构的构型及其宏观布局。YUN 等 [17] 针对粘弹性阻尼材料结构，提出了基于微观结构设计的瞬态响应衰减优化方法，显著提升了结构的动态性能。

> 频域响应拓扑优化主要用来改善结构的稳态振动问题 [18-19]。ZHAO 等 [20] 通过构建增强解耦灵敏度分析方法，成功解决了双尺度结构频率响应分析的尺度耦合难题；并进一步开发了一种自适应混合展开法 [21]，实现了谐波激励区间内结构拓扑的高效优化。ZHOU 等 [22] 将时域分析方法和频域求解目标相结合，提出了一种时域伴随变量法，实现了跨域灵敏度分析。CAI 等 [23] 针对动态载荷的大小、方向、作用位置和激励频率均存在不确定性问题，建立了动态载荷不确定条件下的周期性结构稳健拓扑优化模型。NIU 等 [24] 系统研究了结构振动响应最小化问题。LIU 等 [25] 针对激励频率高于其固有频率的阻尼结构，提出了基于模态保证准则的激励增频方法；其团队还通过大质量法实现了基础加速度激励向等效力激励的转化 [26, 27]。

→ **段落内部结构**：
1. 句首"然而"承接上段，**先指出空白**（"鲜有涉及动态载荷"）—— 这是关键
2. 紧跟"值得注意的是"段：说明该空白的工程价值（航空航天等）
3. 然后转入"针对时域载荷..."、"频域响应..."两个子方向罗列相关工作

→ **可模仿点**：
- **承接段（第 3 段）的开头先点空白**，比把空白放在末段更高效
- 把"现状 + 空白"逻辑组织：上段讲"双模量"研究多 → 这段讲"双模量+动态"研究少 → 自然引出本文方向
- 用"针对时域..."和"频域响应..."把这一大类工作分成 2 个子话题，每个子话题列 3–5 篇代表作

#### 第 4 段：研究空白 + 本文创新点（约 240 字）

> 上述研究为动态载荷下的拓扑优化提供了重要理论支撑，然而针对双模量材料结构在动态载荷下的拓扑优化设计仍面临诸多挑战。因此，本文提出了一种适用于动态载荷下的双模量材料结构拓扑优化设计框架，主要方法创新总结如下：第一，构建了基于混合应力单元的动力学分析模型。采用混合应力单元离散设计域，有效克服了传统位移单元易产生剪切锁定问题，显著提高了应力的计算精度；第二，建立了考虑正交异性材料的四相材料结构插值模型。在传统双模量材料模型 [8-9] 的基础上，引入正交异性材料模拟单元同时受拉和受压时状态，从而更真实地模拟双模量材料在复杂应力状态下的力学响应；第三，提出了单/多工况下的单元拉/压状态判定准则。开发基于主应力符号和拉/压容限的判定算法，并引入拉/压应变能指标，解决了单/多工况动态载荷作用下单元拉/压属性判定困难问题。

→ **段落内部结构**（"创新点末段"的标准模板）：
1. **承接铺垫**：先肯定上段（"上述研究...提供了重要理论支撑"）
2. **抛出空白**：（"然而...仍面临诸多挑战"）
3. **总览本文工作**：（"因此，本文提出了..."）
4. **创新点列举**："第一... 第二... 第三..."

→ **每条创新点的内部结构**（约 80 字）：
   - 句 1：**主标题陈述**（"构建了 ... 模型"）—— 用句号
   - 句 2：**做法 + 机理收益**（"采用 X，有效克服了 Y，显著提高了 Z"）

→ **可模仿点**：
- "第一""第二""第三"用句号断开，不用顿号或逗号
- 每条创新点先 What（主标题），再 Why（机理收益）—— 而不是反过来
- 创新点之间字数均衡（70–90 字），避免头重脚轻

### A.4 结论（共 3 段，约 540 字）

#### 第 1 段：本文工作 + 主要方法（约 165 字）

> （1）本文提出了一种动态载荷作用下的双模量材料结构拓扑优化方法。采用应力计算更准确地混合应力单元离散设计域，避免了使用传统有限元单元易发生剪切锁定的问题。发展了一种基于单元主应力符号和拉/压容限的单元拉/压状态判断准则，进一步构建了多工况下考虑单元拉/压应变能的单元拉/压状态判断准则。建立了基于四相材料的双模量材料结构优化设计方法。研究了载荷激励频率、约束条件、多工况等对双模量材料结构拓扑优化结果的影响规律。

→ **结构**：以本文工作为统领的整段陈述，**没有明确的(1)(2)(3) 子条**——这是结论的"宏观视角"
→ **可模仿点**：结论第 1 段是"放电影"，让审稿人重温本文工作；不需要严格分点

#### 第 2 段：详细成果（带量化结果，约 320 字）

> （2）通过三个数值算例验证本文所提方法的可行性。①所有的数值算例均得到了边界清晰的拓扑优化结果；②当设置不同的拉/压弹性模量比值时，所得拓扑优化结果完全不同，表明了本文所提方法具有设计不同拉/压弹性模量的双模量复合材料结构的能力；③与静态载荷作用下的拓扑优化结果相比，动态载荷作用下的拓扑优化结果在目标函数上明显大于静态载荷作用下的优化结果；④不同的拉压容限对各种材料的体积分布具有很大的影响...；⑤不同的激励频率影响着优化结果的对称性，激励频率越大，所得优化结果越不对称；⑥多工况载荷作用下的权重的大小同样影响优化结果中材料的分布。

→ **结构**：用 ①②③④⑤⑥ 圈号列**6 条具体观察**，每条 30–60 字
→ **可模仿点**：
- 当本文有多个具体实验观察时，用圈号 ①②③ 比 (1)(2)(3) 更紧凑
- 每条观察都从"现象 → 含义"展开（"当...时，结果...，表明..."）
- 顶刊审稿人喜欢这种"密度高、信息量大"的结论段

#### 第 3 段：不足与展望（约 60 字）

> （3）本文研究侧重于揭示动态载荷下双模量材料结构的优化原理，所构建的单元拉/压状态判断准则针对二维情况。考虑到三维设计域在实际工程中应用更为广泛，下一步将着重研究三维空间结构的双模量材料结构设计方法。

→ **结构**：先**承认局限**（"针对二维情况"），再**给出未来方向**（"下一步...三维空间"）
→ **可模仿点**：
- 不足部分**只 1 条**，**具体化**（不是"还有提升空间"这种空话）
- 展望部分**与不足直接对应**（"二维 → 三维"），有逻辑闭环

### A.5 仿例 A 的整体节奏

```
摘要（280 字）  →  极致紧凑：5 段五段式，每段 1 句或 1 句群
   ↓
引言（1100 字）→  4 段：宏观脉络 → 子话题 1 → 子话题 2+空白 → 创新点
   ↓
方法 / 实验  →  按章节展开（细节略）
   ↓
结论（540 字）→  3 段：宏观回顾 → 详细观察 → 不足展望
```

**节奏特征**：摘要紧凑、引言宽展（占全文 13%）、结论中等。这是顶刊力学方向的典型节奏。

## 仿例 B：张九思《半监督学习视角下序列化变分自编码器驱动的高端装备智能剩余寿命预测》

> 此仿例覆盖深度学习方向，与仿例 A 形成对照。**注意**：仿例 B 引言开头使用"在当今日益智能化和数字化的工业领域..."这种过渡句——这种写法在中文工科顶刊里**仍然常见**，但在 `references/stage-polishing.md` 中我们将其标记为"AI 味/中学议论文味"的开头模板。**审慎学习**：你可以使用类似句式，但更优先的是仿例 A 的"直接定义领域"开头。

### B.1 标题

> **半监督学习视角下序列化变分自编码器驱动的高端装备智能剩余寿命预测**

→ **结构**：「方法视角（半监督学习视角下）」+ 「核心方法（序列化变分自编码器）」+ 「对象+任务（高端装备智能剩余寿命预测）」
→ **字数**：30 字（**偏长**，超出 16–28 字的舒适区）
→ **可模仿点**：当你的方法包含"视角 + 模型"两层时，这种结构能讲清；但要警惕字数膨胀，可考虑去掉"智能"等修饰

### B.2 摘要（约 320 字）

> 剩余寿命预测是保障高端装备安全性与可靠性的核心内容。

→ **【背景】1 句 23 字**：极简的领域定义

## 仿写步骤建议

1. **先读两篇仿例的全文**（不只是摘要/引言/结论），感受整体节奏
2. **照仿例的字数分配**列出你论文的"字数预算"
3. **照仿例的段落结构**列出每段的"功能"和"目标字数"
4. **写第一稿时**：把仿例对应段落放在你 Word/编辑器**右侧**，**句式参照**写
5. **第一稿写完后**：去除仿例里的具体术语，确保你的内容是**自己的研究**而非借词
6. **润色阶段**：对照本 reference 的"可模仿点"清单，确认你写的版本是否具备同等的"信息密度"和"节奏"

⚠️ **抄袭红线**：可以借鉴**结构、句式、用词习惯**；不可借鉴**具体内容、研究结论、实验数据**。学术写作的"仿"和"抄"边界清晰：
- ✅ "采用 X 单元离散设计域，以获得更准确的应力计算结果" —— 句式
- ❌ "采用混合应力单元离散设计域..." —— 这是原作者的方法，照搬即抄袭
