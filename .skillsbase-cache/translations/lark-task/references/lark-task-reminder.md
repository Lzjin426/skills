# 任务 +提醒

> **前提条件：** 请阅读 `../lark-shared/SKILL.md` 以了解身份验证、全局参数和安全规则。
> **优先级：** 对于创建或修改任务提醒时间，请优先使用此 `+reminder` 快捷方式，而非其他任务更新方法。它提供了更可靠、更直接的方式来管理提醒。

管理任务提醒。设置新提醒或移除现有提醒。请注意，设置任务提醒需要任务有截止日期。

## 推荐命令

```bash
# 设置提醒（例如，截止前 30 分钟）
lark-cli task +reminder --task-id "<task_guid>" --set "30"

# 设置提醒（例如，截止前 1 小时）
lark-cli task +reminder --task-id "<task_guid>" --set "1h"

# 移除所有提醒
lark-cli task +reminder --task-id "<task_guid>" --remove "true"
```

## 参数

| 参数 | 是否必需 | 描述 |
|-----------|----------|-------------|
| `--task-id <guid>` | 是 | 要修改的任务 GUID。对于飞书任务应用链接，请使用 `guid` 查询参数，而不是 `suite_entity_num` / 显示任务 ID（如 `t104121`）。 |
| `--set <val>` | 否 | 相对于截止时间的触发分钟数。支持数字（例如 `30`）或带单位的值（例如 `15m`、`1h`、`1d`）。 |
| `--remove <bool>` | 否 | 如果设置为 `true`，则移除任务的所有现有提醒。 |

## 工作流程

1.  确认任务和提醒操作。
2.  执行命令。
3.  报告成功。

> [!CAUTION]
> 这是一个 **写入操作** —— 执行前必须确认用户意图。
