# 任务 +reopen

> **前提条件：** 请阅读 `../lark-shared/SKILL.md` 以了解身份验证、全局参数和安全规则。

重新打开一个先前已完成的任务。

## 推荐命令

```bash
# 重新打开一个任务
lark-cli task +reopen --task-id "<task_guid>"
```

## 参数

| 参数 | 是否必需 | 描述 |
|-----------|----------|-------------|
| `--task-id <guid>` | 是 | 要重新打开的任务 GUID。对于飞书任务应用链接，请使用 `guid` 查询参数，而不是 `suite_entity_num` / 显示的任务 ID（如 `t104121`）。 |

## 工作流程

1. 确认要重新打开的任务。
2. 执行命令。
3. 报告成功。

> [!CAUTION]
> 这是一个 **写入操作** —— 执行前必须确认用户意图。
