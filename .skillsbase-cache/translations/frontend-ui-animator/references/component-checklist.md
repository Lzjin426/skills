# 组件动画检查清单

使用此模板来审计和规划项目的动画效果。

---

## 项目分析模板

### 1. 项目结构扫描

```bash
# 需要分析的页面
app/
├── page.tsx              # 首页
├── about/page.tsx        # 关于页面
├── services/page.tsx     # 服务页面
└── contact/page.tsx      # 联系页面

# 需要分析的组件
components/
├── ui/                   # 基础 UI 组件
├── sections/             # 页面区块
└── layout/               # 布局组件
```

### 2. 技术设置检查

| 项目 | 状态 | 备注 |
|------|--------|-------|
| Tailwind 配置包含关键帧 | ☐ | 检查 `theme.extend.keyframes` |
| Tailwind 配置包含动画 | ☐ | 检查 `theme.extend.animation` |
| 已安装 Framer Motion | ☐ | 检查 `package.json` |
| 已安装 GSAP | ☐ | 检查 `package.json` |
| 存在减少运动的 CSS | ☐ | 检查 `globals.css` |

## 动画审计表

### 优先级 1：首屏 / 第一印象

| 组件 | 当前状态 | 建议动画 | 触发条件 | 工作量 |
|-----------|--------------|-------------------|---------|--------|
| 首屏标题 | 静态 | 淡入 + 向上滑动，逐字出现 | 加载 | 低 |
| 首屏副文本 | 静态 | 延迟淡入 | 加载 | 低 |
| 首屏行动号召按钮 | 静态 | 淡入 + 悬停上浮 | 加载/悬停 | 低 |
| 首屏图片 | 静态 | 缩放进入或 Ken Burns 效果 | 加载 | 中 |
| 首屏背景 | 静态 | 渐变切换或视差 | 加载/滚动 | 中 |

### 优先级 2：悬停交互

| 组件 | 当前状态 | 建议动画 | 触发条件 | 工作量 |
|-----------|--------------|-------------------|---------|--------|
| 导航链接 | 基础悬停 | 下划线滑动或颜色变化 | 悬停 | 低 |
| 按钮（全部） | 基础悬停 | 上浮 + 阴影 | 悬停 | 低 |
| 卡片 | 基础悬停 | 缩放 + 阴影 | 悬停 | 低 |
| 卡片内图片 | 静态 | 父级悬停时缩放 | 悬停 | 低 |
| 社交图标 | 基础悬停 | 缩放 + 颜色变化 | 悬停 | 低 |

### 优先级 3：内容揭示（滚动）

| 组件 | 当前状态 | 建议动画 | 触发条件 | 工作量 |
|-----------|--------------|-------------------|---------|--------|
| 区块标题 | 静态 | 淡入 + 向上滑动 | 滚动 | 中 |
| 功能卡片 | 静态 | 交错淡入 | 滚动 | 中 |
| 客户评价 | 静态 | 从两侧滑入 | 滚动 | 中 |
| 统计数据/数字 | 静态 | 数字递增动画 | 滚动 | 高 |
| 图片 | 静态 | 淡入 + 缩放 | 滚动 | 中 |

### 优先级 4：背景 / 氛围

| 组件 | 当前状态 | 建议动画 | 触发条件 | 工作量 |
|-----------|--------------|-------------------|---------|--------|
| 页面背景 | 纯色 | 渐变网格 | 始终 | 低 |
| 区块分隔线 | 静态 | 波浪或曲线 | 始终 | 低 |
| 装饰元素 | 静态 | 浮动 | 始终 | 低 |
| 噪点/颗粒叠加层 | 无 | 细微纹理 | 始终 | 低 |

### 优先级 5：导航 / 过渡

| 组件 | 当前状态 | 建议动画 | 触发条件 | 工作量 |
|-----------|--------------|-------------------|---------|--------|
| 导航栏 | 静态 | 基于滚动的模糊/阴影 | 滚动 | 中 |
| 移动端菜单 | 即时显示/隐藏 | 从右侧滑入 | 点击 | 中 |
| 页面过渡 | 无 | 淡入淡出或裁剪揭示 | 路由切换 | 高 |
| 滚动至区块 | 瞬时跳转 | 平滑滚动 | 点击 | 低 |

## 实施阶段

### 阶段 1：快速见效（1-2 小时）

专注于无需新依赖的纯 CSS 动画：

- [ ] 向 Tailwind 配置添加基础关键帧
- [ ] 向 globals.css 添加减少运动支持
- [ ] 首屏区块淡入动画
- [ ] 按钮悬停效果（所有按钮）
- [ ] 卡片悬停效果（缩放 + 阴影）
- [ ] 导航栏滚动效果

### 阶段 2：滚动揭示（2-3 小时）

添加基于 IntersectionObserver 的动画：

- [ ] 创建 `useScrollReveal` 钩子
- [ ] 区块标题揭示
- [ ] 功能/服务卡片交错揭示
- [ ] 客户评价揭示
- [ ] 页脚内容揭示

### 阶段 3：增强效果（3-4 小时）

更复杂的动画：

- [ ] 首屏文本字母/词动画
- [ ] 卡片悬停时图片缩放
- [ ] 徽标/合作伙伴跑马灯效果
- [ ] 卡片上手电筒效果（如适用）
- [ ] 移动端菜单滑动动画

### 阶段 4：优化（可选，2+ 小时）

需要更多工作量的高级效果：

- [ ] 页面过渡（Framer Motion）
- [ ] 数字递增动画
- [ ] 视差背景
- [ ] 行动号召按钮上的边框光束效果
- [ ] 自定义光标效果

## 动画令牌参考

在整个项目中使用一致的时序和缓动：

| 令牌 | 值 | 使用场景 |
|-------|-------|----------|
| `duration-fast` | 150ms | 微交互，悬停 |
| `duration-normal` | 300ms | 标准过渡 |
| `duration-slow` | 500ms | 揭示，进入 |
| `duration-slower` | 800ms | 首屏动画 |
| `ease-out` | cubic-bezier(0, 0, 0.2, 1) | 进入 |
| `ease-in-out` | cubic-bezier(0.4, 0, 0.2, 1) | 连续动画 |
| `ease-spring` | cubic-bezier(0.34, 1.56, 0.64, 1) | 弹性效果 |

## 质量检查清单

在标记动画实施完成之前：

- [ ] 所有动画都尊重 `prefers-reduced-motion`
- [ ] 动画未导致布局偏移（CLS）
- [ ] 动画不阻塞用户交互
- [ ] 交错延迟合理（不太慢）
- [ ] 悬停状态具有适当的过渡时长
- [ ] 移动端性能可接受（在真实设备上测试）
- [ ] 动画增强内容，而非分散注意力
