# 研究结果（2026-02-25）

源基准：`Desktop/Fullstop_base/multi_agent_debate_deliberation_research_2026-02-25.html`

## 1) 适用边界

仅在以下三个条件同时满足时启用辩论式多智能体审议：

1. 存在可辩论空间。
2. 存在可测试的评判标准。
3. 存在可证伪的证据链。

否则，辩论通常只会增加成本而无法提升质量。

## 2) 关键证据摘要

1. MAD（Du 等人，2023）：在多项任务上相比单智能体有显著提升。
2. 稀疏拓扑（Li 等人，2024）：在某些任务中，保持或提升质量的同时降低了令牌成本（在特定设置中报告降低 40% 以上）。
3. ChatEval（Chan 等人，2023）：多智能体讨论在评审任务中提升了与人类评估的一致性。
4. BioNLP 2025 医疗场景：辩论框架优于部分基线，但成本显著更高。
5. EMNLP Findings 2025：辩论可能在没有明确控制的情况下放大偏见。

## 3) 可操作建议

1. 默认进行 2 轮，最多 3 轮。
2. 强制使用证据检查器以减少自信的错误。
3. 在评审阶段使用固定的评分标准。
4. 设置预算和停止阈值以防止无限循环。

## 4) 建议默认设置

1. 角色：2 名提议者 + 1 名批评者 + 1 名证据检查员 + 1 名评审员。
2. 轮次：2 轮（第 3 轮需严格把关）。
3. 停止规则：连续两个检查点增益 <1% 或无新证据。
4. 高风险任务：允许“证据不足”作为最终状态。

## 5) 残留风险

1. 共享模型偏见可能导致自信但错误的共识。
2. 角色崩溃可能削弱辩论的价值。
3. 引用规范不佳可能造成虚假的稳健性。

在启用此技能前，务必运行把关检查。
