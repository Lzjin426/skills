# 简介：README

## 何时使用

此简介适用于仓库、项目、工具或软件包的引导文档。

## 读者期望

读者希望了解这是什么、为什么重要，以及如何快速上手。

## 默认结构

- 项目概述
- 快速开始或安装指南
- 基本用法
- 重要限制、提示或后续链接

## 默认载体

混合形式，通常以段落为主，辅以简短的代码块和列表。

## 默认处理项

- 研究：`关闭`
- 认知路径：`关闭`
- 插图：`可选`
- 人性化：`可选`

## 样式默认值

- 以引导和快速上手价值为首要内容
- 若已有仓库样式，则沿用现有风格
- 除非项目本身已使用，否则不强制要求编号标题或顶层分隔符
