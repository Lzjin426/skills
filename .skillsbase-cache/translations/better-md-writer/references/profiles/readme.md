# 简介：README

## 何时使用

此简介适用于仓库、项目、工具或包的引导文档。

## 读者预期

读者希望了解这是什么、为什么重要，以及如何快速上手。

## 默认结构

- 项目简介
- 快速开始或安装指南
- 基本用法
- 重要限制、提示或后续链接

## 默认载体

混合形式，通常以段落为主，辅以简短的代码块和列表。

## 默认处理

- 研究：`关闭`
- 认知路径：`关闭`
- 示例说明：`可选`
- 人性化表达：`可选`

## 样式默认值

- 以引导和快速上手价值为首要内容
- 若仓库已有样式，则遵循现有风格
- 除非项目本身已使用，否则不强制使用编号标题或顶级分隔符
