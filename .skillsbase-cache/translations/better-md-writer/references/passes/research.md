# 更优 Markdown 写作助手：研究功能

研究功能为可选，默认设置为 `关闭`。

## 级别

- **关闭**
  - 当用户已提供所需材料，或文档主要为记录性质时使用。

- **轻度**
  - 适用于术语核对、简要验证或确认当前最佳实践。
  - 保持搜索范围集中且有明确目的。

- **深度**
  - 适用于选项对比、需要大量证据的写作、时效性强的主题，或用户明确要求进行研究时。
  - 总结发现、指出矛盾之处，并在适当时引用来源。

## 何时启用

仅当满足以下任一条件时启用研究功能：
- 主题随时间变化
- 文档涉及方案比较
- 读者需要证据或权威参考
- 用户明确要求进行研究

## 使用准则

- 若用户的一手材料已能完成任务，应优先使用
- 不应仅为增加文档篇幅而进行搜索
- 将研究发现融入文档自身的表达风格中
- 对不确定之处进行标注，而非隐藏
