# 样本文献模式提炼

语料来自 5 篇中文工科论文，其中 1 篇偏综述型，4 篇偏方法型。结论：不能只学“引多少篇”，要学“哪类句子必须有证据”。

## 总体规律

1. 文献综述不一定单独成章。

- 综述型文章会在前言后继续分节综述，并在各小节持续高密度引用。
- 方法型文章通常把综述嵌在引言中，结构常见为：背景 -> 既有路线 -> 代表工作 -> 不足 -> 本文。

2. 段落组织通常遵循“总述一句 + 分述几句 + 缺口一句”。

- 总述句常配 grouped cites，如 `[5-7]`、`[25-28,31]`。
- 分述句常按“作者/模型 + 动作 + 结果/特点 + 引文”书写。
- 缺口句往往绑定前文路线或代表工作，不能脱离对象空泛批评。

3. 引文密度受文体控制，而不是平均分配。

- 综述型文章：高密度、连续分布、同一段多处落引。
- 方法型引言：中高密度，集中在背景、分类、代表工作、不足判断。
- 方法章节：低到中密度，只在理论来源、模块对比、设计动机处落引。
- 实验章节：低密度，只在基线、数据集、评价指标、来源说明处落引。
- 结论：通常不再密集补引。

## 五篇样本的可复用模式

### 样本 1：综述型

- 类型：技术研究现状与装备变革综述。
- 特点：前言中先交代制造背景与技术变迁，再在正文各节持续做“技术分层综述”。
- 可复用点：
  - 用“历史路线切换”组织综述，如“化学铣切 -> 单机铣削 -> 镜像铣削”。
  - 用 grouped cites 支撑路线总述。
  - 每个子方向内部采用“基础理论/结构设计/控制策略/挑战展望”的固定展开。

### 样本 2：方法型

- 类型：超声导波损伤评估方法。
- 前言模式：
  - 背景与任务价值
  - 路线 A：离散式传感网络
  - 路线 B：全波场导波检测
  - 各路线优缺点
  - 本文方法与目标
- 可复用点：
  - 两条或多条技术路线并列综述后，再写“精度与成本难折中”的缺口句。
  - 缺口后立刻接“为此/针对以上问题，本文……”。

### 样本 3：方法型

- 类型：工件表面图像超分辨率。
- 前言模式：
  - 工业任务背景
  - 传统方法
  - CNN 路线
  - Transformer 路线
  - 面向工业场景的二次问题定义
  - 本文模块设计
- 可复用点：
  - 先讲通用视觉领域，再缩到工业场景。
  - 不足句不是空泛说“效果不好”，而是指向“高频特征、层次依赖、计算开销”。

### 样本 4：方法型

- 类型：锂电池低温复合加热。
- 前言模式：
  - 应用背景
  - 一级分类：外部加热 / 内部加热
  - 二级分类：直流 / 交流 / 自加热电池
  - 各方案优缺点
  - 复合方案动机
- 可复用点：
  - 适合做“分层分类综述”。
  - 分类总述用 grouped cites；具体方法用单篇或少量文献支撑。

### 样本 5：方法型

- 类型：WTA 优化算法。
- 前言模式：
  - 问题定义与应用价值
  - 相关工程优化背景
  - 算法谱系：NSGA-II -> NSGA-III -> 加速策略
  - 现有策略挑战
  - 本文贡献列表
- 可复用点：
  - 适合算法论文：先问题，再算法谱系，再指出“数据、计算、参数”三类瓶颈。
  - 贡献列表前通常有一段“现有工作重质量、轻时间”的收束句。

## 后文章节的补引规律

1. 方法章节

- 引经典理论、标准模块、已有算法时补引。
- 对本文公式推导、模块流程、参数定义通常不补引。
- “为解决上述问题，本文设计……”一般不引。

2. 实验章节

- 数据集、评价指标、对比基线首次出现时补引。
- 对“表 1 显示本文优于其他方法”这类本研究结论通常不引。

3. 结论章节

- 结论主要回收本文工作与结果，通常不做高密度补引。

## 直接可用的判断

- 如果用户要写“研究现状”小节，优先参考样本 1 的高密度模式。
- 如果用户要润色方法论文引言，优先参考样本 2~5 的“背景-路线-缺口-本文”模式。
- 如果用户只是补方法段或实验段的引用，不要强行套用综述段密度。
